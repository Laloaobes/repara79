# REPARA - 79: Sistema de Gestión de Tickets de Mantenimiento

## 📝 Descripción General
REPARA-79 es una plataforma web integral para optimizar el flujo de reporte y reparación de desperfectos institucionales.

### 🎯 Objetivos Principales
* Trazabilidad Visual con evidencias.
* Control de Insumos y costos.
* Soberanía y Seguridad On-Premise.

### 💻 Stack Tecnológico
* Frontend: React 18+, Vite, Tailwind.
* Backend: API REST Laravel 11.
* Base de Datos: PostgreSQL 16.
* Servidor: Ubuntu Server con Docker Compose.
