# 🔄 Flujo de Trabajo de Git (GitFlow Adaptado)

## 🌿 Ramas Principales
* main: Código de producción estable.
* develop: Rama de integración compartida.

## 🚀 Ramas de Características
Cada integrante trabaja en ramas independientes partiendo de develop (ej. feat/ o style/).

## 🔀 Proceso de Integración
Se sube la rama característica a GitHub y se abre un Pull Request hacia develop para revisión.
